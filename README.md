# Nemerion Dog Design - Landing Page

This is a premium-looking, responsive landing page for a luxury dog collar brand.

## Features
- **Elegant design:** Double gold frame, corner decorations, and Cinzel typography.
- **Interactive gallery:** Instagram-style collection view with hover effects.
- **Responsive layout:** Displays perfectly on both mobile and desktop devices.

## Technologies
- HTML5
- CSS3 (Flexbox, Grid, Mask-image)
- Google Fonts API
